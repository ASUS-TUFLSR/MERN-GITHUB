# Github-finderV2 
- V2 Of Github-Finder

# Features
- Login & Signup page for (auth) Using PassportJS via Github SignIN / SignUp
- Explore Page To Popular Repositories
- Likes Page To Like Profiles
- Mobile-first responsive layout


# Quick start
- Clone this repo - "https://github.com/ASUS-TUFLSR/MERN-GITHUB.git"
- Install dependencies:
 - npm install && cd frontend npm install
- SetUp All the ENV variables
- Run dev server:
- npm run dev

# Build for production:
- npm run build
- Preview production build:
  - npm run preview

# Deployment
- You can deploy on Render, Vercel etc..
